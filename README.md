GraphRAG Offline Knowledge Portal (Demo)

This project implements a fully offline GraphRAG pipeline that can read PDF documents, extract knowledge, build a knowledge graph, and perform hybrid retrieval (graph + vector search) to find the most relevant information for any user question â€” without using any cloud APIs.

This version does NOT require an LLM.
It only retrieves relevant evidence chunks.
(LLM answer generation can be added later.)



# Installation

1. Install Python dependencies
pip install spacy networkx sentence-transformers faiss-cpu pypdf python-dotenv

2. (Optional) OCR Support for Scanned PDFs
pip install pdf2image pytesseract pillow

3. Download spaCy English model
python -m spacy download en_core_web_sm

# Running the Demo

Add your PDFs into the docs/ folder (any number of PDFs).
Run the script:

python graphrag_full_demo.py



Once loaded, you can type:

Question: What is the engine specification?

The system returns the top relevant chunks, along with their document names, chunk IDs, and vector distances:

#1 IDX:12 DOC:manual.pdf CHUNK_ID:3 DIST:0.934
